# Navarre — Kodi repository

A self-hosted [Kodi](https://kodi.tv) repository for **Navarre**, a personal fork of
*Arctic: Zephyr – Reloaded*.

## Install
1. In Kodi: **Settings → Add-ons → Install from zip file** → install
   [`repository.navarre-1.0.0.zip`](zips/repository.navarre/repository.navarre-1.0.0.zip)
   (enable *Unknown sources* first if prompted).
2. Then **Install from repository → Navarre Repository → Look and feel → Skins → Navarre**.
3. Updates arrive automatically.
